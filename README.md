# Parkinson Demographie-Analyse

Dieses Projekt untersucht einen demographischen Datensatz zur Parkinson-Erkrankung. Die Analyse kombiniert programmatische Datenzeilen in Python mit einer interaktiven visuellen Aufbereitung in Power BI.

## 📁 Projektstruktur

Das Repository ist wie folgt aufgebaut:

* **`[NAME_DEINES_NOTEBOOKS].ipynb`** – Das Jupyter Notebook mit der vollständigen Python-Analyse.
* **`[NAME_DER_DASHBOARD_DATEI].pdf`** – Ein Export des Power BI Dashboards zur visuellen Exploration.
* **`[NAME_DER_DATEN_DATEI].xls`** – Der zugrundeliegende demographische Datensatz.
* **`[NAME_DER_FRAGESTELLUNG].pdf`** – Die zugrundeliegenden Leitfragen und Ziele dieser Analyse.

## 🚀 Analyse-Schwerpunkte

* **Fragestellung** – Systematische Beantwortung der vorgegebenen Kernfragen aus dem PDF-Dokument.
* **Explorative Datenanalyse (EDA)** – Statistische Auswertung von Alter, Geschlecht und demographischen Faktoren in Python.
* **Interaktives Dashboard** – Visuelle Aufbereitung der Ergebnisse in Power BI für schnelle Erkenntnisse.

## 🛠️ Technologien & Tools

* **Python 3.x** & **Jupyter Notebook** (Pandas, NumPy, Matplotlib, Seaborn)
* **Microsoft Power BI** (Datenvisualisierung & Dashboard-Design)
* **Microsoft Excel** (Datenbasis im `.xls`-Format)

## 💻 Ausführung der Python-Analyse

1. **Repository klonen**
   ```bash
   git clone https://github.com
   ```
2. **Abhängigkeiten installieren**
   ```bash
   pip install pandas xlrd matplotlib seaborn notebook
   ```
   *(Hinweis: `xlrd` wird benötigt, um ältere `.xls`-Dateien in Python zu öffnen)*
3. **Notebook starten**
   ```bash
   jupyter notebook
   ```
